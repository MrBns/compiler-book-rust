# Getting Started

## Welcome!

Hello! Welcome to **Compiler Book in Rust** 🦀

This book will teach you how to build a real **compiler** from scratch using the **Rust programming language**.

Do not worry if you are new to compilers. We will learn everything step by step.

> **This book is for you if:**
> You know basic Rust (variables, functions, structs, enums, match).
> You want to learn how compilers work.
> You want a fun project to practice Rust.
> English is not your first language — we use simple words here!

## What Will We Build?

We will invent our own small language. We call it **Pico**.

Here is a sample Pico program:

```
let name = "Alice";
let age  = 25;

if age > 18 {
    print("Adult");
} else {
    print("Not adult");
}

fn add(a, b) {
    return a + b;
}

let result = add(10, 20);
print(result);
```

Our compiler will read this Pico code and output working **TypeScript** code:

```typescript
// Generated by Pico compiler
const name = "Alice";
const age = 25;

if ((age > 18)) {
    console.log("Adult");
} else {
    console.log("Not adult");
}

function add(a: any, b: any): any {
    return (a + b);
}

const result = add(10, 20);
console.log(result);
```

Pretty cool, right? Let's get started!

## Set Up Your Rust Project

First, make sure you have Rust installed. Open your terminal and run:

```bash
# Check if Rust is installed
rustc --version
cargo --version
```

If you see version numbers, you are good. If not, go to [https://rustup.rs](https://rustup.rs) and install Rust.

Now create a new project:

```bash
# Create a new Rust project called "pico"
cargo new pico
cd pico
```

You will see this folder structure:

```
pico/
├── src/
│   └── main.rs    ← our code goes here
└── Cargo.toml     ← project settings
```

Open `src/main.rs` and replace everything with:

```rust
// main.rs — Entry point of our Pico compiler

fn main() {
    println!("Pico compiler is alive! 🦀");
}
```

Run it:

```bash
cargo run
```

You should see:

```
Pico compiler is alive! 🦀
```

## Project Structure

As we build our compiler, we will split the code into several files. Here is what the final structure will look like:

```
pico/
├── src/
│   ├── main.rs         ← glues everything together
│   ├── token.rs        ← token type definitions
│   ├── lexer.rs        ← turns text into tokens
│   ├── ast.rs          ← defines the tree types
│   ├── parser.rs       ← turns tokens into a tree
│   ├── semantic.rs     ← checks for mistakes
│   └── codegen.rs      ← generates TypeScript output
└── Cargo.toml
```

We will build each file one by one in this book.
